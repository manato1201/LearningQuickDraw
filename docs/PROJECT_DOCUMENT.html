固定小数点 (16.16) の仕組み

32bit 整数 = 上位 16bit (整数部) + 下位 16bit (小数部)

3.5 → 3 * 65536 + 32768 = 229376 = 0x00038000

float DDA:   fy += slope;       round(fy)   ← float 加算 + round()
固定小数点:  fy += slope_fixed; fy >> 16    ← 整数加算 + シフト (高速)
